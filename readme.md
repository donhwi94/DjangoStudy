# README

- `chinook.db`를 활용한 `Restful`한 `Django` 만들어보기

## How to Execute

- `Develop`
    - `python manager.py runserver --settings=conf.setting.develop`

## Requirement

- `requirement.txt` 참조

## Dependency
 - `Framework`
   - DjangoRestFramework
   - DRF-YASG
 - DB (SQLite)
